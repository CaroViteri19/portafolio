Acerca de mí

Desarrolladora Full-Stack con experiencia en gestión de proyectos educativos y tecnológicos. Combino habilidades en Java, Spring Boot, React y SQL con un enfoque en calidad de software y optimización de procesos. He trabajado en aplicaciones web como Ciempies, integrando bases de datos relacionales, módulos de asistencia y formularios dinámicos para mejorar la inclusión y la gestión escolar.

Proyectos Destacados

Es un sistema de gestión interna desarrollado como práctica académica. Organiza y administra módulos funcionales como usuarios, roles y reportes.

Es una aplicación web que permite al usuario seleccionar y visualizar diferentes estados de ánimo mediante un diseño interactivo y dinámico.

GuitarBlog

Es una aplicación web tipo blog desarrollada como práctica de diseño con Bootstrap 5, enfocada en el mundo de la música y la guitarra.
